C99 Documentation

Introduction

C99 is an informal name for the standard for the C programming language as of 1999. It was developed by the American National Standards Institute (ANSI). The C99 standard aims to provide a more comprehensive and consistent specification for the C language, which is widely used in system programming and other applications.

Variables

Variables are used to store data in a program. In C99, variables must be declared before they are used. The declaration of a variable specifies its type and name. For example, the following code declares an integer variable named x: 

                
                int x;

In this example, the variable x is declared as an integer using the int keyword. The variable can then be assigned a value using the assignment operator (=). For example: 

                
                x = 42;

Data Types

C99 provides several built-in data types for storing different kinds of data. The most common data types include:

  • int: Integer data type for storing whole numbers.
  • float: Floating-point data type for storing decimal numbers.
  • char: Character data type for storing single characters.

In addition to these basic data types, C99 also provides modifiers that can be used to modify the size and signedness of the data types. For example, the long modifier can be used to create a long integer data type, and the unsigned modifier can be used to create an unsigned integer data type.

Operators

Operators are symbols that are used to perform operations on variables and values. C99 provides a wide range of operators for performing arithmetic, logical, and bitwise operations. Some of the most common operators include:

  • + : Addition operator
  • - : Subtraction operator
  • * : Multiplication operator
  • / : Division operator
  • % : Modulus operator
  • && : Logical AND operator
  • || : Logical OR operator
  • ! : Logical NOT operator
  • & : Bitwise AND operator
  • | : Bitwise OR operator
  • ^ : Bitwise XOR operator
  • << : Left shift operator
  • >> : Right shift operator

Control Statements

Control statements are used to control the flow of a program. C99 provides several control statements for making decisions and repeating code. Some of the most common control statements include:

  • if statement: Executes a block of code if a specified condition is true.
  • else statement: Executes a block of code if the same condition is false.
  • while statement: Executes a block of code while a specified condition is true.
  • for statement: Executes a block of code a specified number of times.
  • switch statement: Executes a block of code based on the value of a variable.

Functions

Functions are blocks of code that can be called from other parts of a program. In C99, functions are declared using the following syntax: 

                
                return_type function_name(parameter1, parameter2, ...);

For example, the following code declares a function named add that takes two integer parameters and returns an integer: 

                
                int add(int x, int y);

The function can then be defined using the following syntax:

                
                int add(int x, int y) {
                    return x + y;
                }

The function can be called from other parts of the program using the function name and passing the required parameters. For example: 

                
                int result = add(3, 4);

Preprocessor Directives

Preprocessor directives are special commands that are processed by the C preprocessor before the compilation of the program. C99 provides several preprocessor directives for including header files, defining constants, and performing other tasks. Some of the most common preprocessor directives include:

  • #include: Includes a header file in the program.
  • #define: Defines a constant value or a macro.
  • #ifdef: Checks if a macro is defined.
  • #ifndef: Checks if a macro is not defined.
  • #endif: Ends an #ifdef or #ifndef block.
